# Zig Invaders

A Space Invaders clone written in Zig.

## Prerequisites

- [Zig](https://ziglang.org/download/) (latest stable or master)

## Building

To build the project, run:

```bash
zig build
```

The executable will be located in `zig-out/bin/`.

## Running

To build and run immediately:

```bash
zig build run
```
